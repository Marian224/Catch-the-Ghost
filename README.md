# 👻 Catch the Ghost! (Лови Привида)

Ласкаво просимо до **Catch the Ghost!** — динамічної та веселої браузерної міні-гри, де твоє головне завдання — впіймати спритного привида, який постійно телепортується по ігровій арені. З кожним успішним кліком привид стає швидшим, а гра — складнішою!

---

## 🎮 Демонстрація гри

* **Жанр:** Аркада / Клікер
* **Тематика:** Spooky Neon (Містичний неон)
* **Технології:** HTML5, CSS3, JavaScript (Vanilla JS)

---

## ✨ Особливості гри

* **Динамічна складність:** Швидкість появи привида збільшується (інтервал зменшується на 60 мс) після кожного успішного кліку.
* **Система життів:** Гравець має 3 життя. Якщо не встигнути клікнути на привида до його наступного переміщення — втрачається одне життя.
* **Збереження рекорду (High Score):** Гра відстежує твій найкращий результат протягом поточної ігрової сесії.
* **Сучасний Spooky-дизайн:** Стильне неонове оформлення, темна палітра кольорів та ефект світіння.
* **Анімації:** * Ефект легкого погойдування (`float`), коли привид активний.
  * Ефект стискання (`caught`), що візуально підтверджує успішне натискання.

---

## 🛠️ Як грати

1. Натисни кнопку **Start Game 🔄**, щоб запустити ігровий цикл.
2. Швидко клікай по привиду 👻, який з'являється у випадкових місцях на синій арені.
3. Не зволікай! Якщо привид зникне до того, як ти на нього натиснеш, ти втратиш серденько (життя).
4. Гра завершується, коли лічильник життів досягне `0`. Спробуй побити свій рекорд!

---

## 📁 Структура проекту

Проект має максимально просту та чисту структуру:

* `index.html` — розмітка сторінки, структура ігрової арени та елементів інтерфейсу.
* `style.css` — стилізація інтерфейсу, адаптивна верстка (Flexbox), анімації появи та зникнення привида.
* `script.js` — логіка гри (ігровий цикл, обробка кліків, випадкове позиціонування, підрахунок очок та життів).

---

## 🚀 Як запустити проект локально

Тобі не потрібні жодні сервери чи додаткові інструменти. Просто виконай ці кроки:

1. Завантаж файли `index.html`, `style.css` та `script.js` в одну папку.
2. Двічі клікни на файл `index.html`, щоб відкрити його у твоєму улюбленому веб-браузері.
3. Насолоджуйся грою!

---

## 💻 Використані технології та концепції JS

* **DOM-маніпуляції:** Динамічне оновлення тексту рахунку, рекорду та життів за допомогою `innerText`.
* **Ігрова петля (Game Loop):** Реалізована через `setInterval` та `clearInterval` для контролю часу між появою привида.
* **Абсолютне позиціонування:** Використання властивостей `style.left` та `style.top` разом із `Math.random()` для точного розміщення привида всередині арени.
* **Обробка станів:** Додавання та видалення CSS-класів (`active`, `caught`) через `classList` для керування анімаціями.
